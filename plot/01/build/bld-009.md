# 基礎計画 第1章 第9話

- 謁見の間の様子
  - 厳粛にして荘厳な武威を湛えた騎士が両側の壁際に並ぶ
  - 子爵の前に跪く四人

- ディックスはアルティバ子爵の三男で鼻つまみ者で無頼漢とつるんで悪さをしていて子爵が対応した
- 魔力の被害はディックスが持っていた魔導具が原因だと判断された
  - この話を聞いたときエクトが名乗り出ようとしたがキスカとフェリが止めた
  - 子爵は褒美を出すと言っていて三人はそれぞれの罪悪感を持った
  - 今は取り調べが済んだ翌日、午前の中ほど

- logline = 子爵は魔導具のことで話を聞きたい　魔力氾濫は不本意なことのようだったから、それより力を手に入れたところに関心がある
  - 屍蝋の闇炎
  - 魔法の塔について知っているか尋ねる

- テイニィと昼食を摂ったあと広場の一画で
  - 宝玉
    - エクトの知る紋章の描かれたアイテム
      - ゼレストラ王国の三つに分かれた紋章 => 太陽、月、天空
        - 三つに分かれる前の紋章
        - 四つの具体的な形象は
          - 天空 = 大きな円の淵を上側には山並み、下側には海の波が描かれていて中央部に交わる百合
          - 太陽 = 円の周りに放射線状に配置された十字になった剣の具象
          - 月 = 三日月の円の中心に魔法陣
      - 手に収まるほどの宝珠
  - 神殿遺跡へ翌日
  - テイニィとは別れる
